Types:CSS Element Selector:

CSS Element Selector element selector selects the HTML element by name
CSS Id Selector:

The id selector selects the id attribute of an HTML element to select a specific element
always unique within the page so it is chosen to select a single, unique element.

CSS Class Selector: ------------------- The class selector selects html elements with a specific class attribute. It is used with a period character .

(full stop symbol) followed by the class name. A class name should not be started with a number.
-- CSS Universal Selector: ----------------------- The universal selector is used as a wildcard character. It selects all the elements on the pages. -- CSS Group Selector: ------------------ The grouping selector is used to select all the elements with the same style definitions. Grouping selector is used to minimize the code. Commas are used to separate each selector in grouping.
